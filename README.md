# Motion & Code

Source code for short animation demos published on the Motion & Code YouTube channel.

Each folder contains a small, self-contained Flutter or SwiftUI animation built for learning, inspiration, and short-form demo videos.

## Flutter

- [Dynamic Island Clone](flutter/dynamic_island_clone) - A pill-shaped live activity that expands into a flight tracking card.
