# Motion & Code

Source code for short animation demos published on the Motion & Code YouTube channel.

Each folder contains a small, self-contained Flutter or SwiftUI animation built for learning, inspiration, and short-form demo videos.

## Flutter

- [Dynamic Island Clone](flutter/dynamic_island_clone) - A pill-shaped live activity that expands into a flight tracking card.
- [App Store Download Button](flutter/app_store_download_button) - A native-style download button that moves through get, loading, progress, installed, and open states.
- [Wallet Card Stack](flutter/wallet_card_stack) - An Apple Wallet-style card stack with expansion and focused card details.
- [Theme Switcher](flutter/theme_switcher) - A dramatic day/night toggle with sun, moon, stars, clouds, and reactive UI.
- [Infinite Carousel](flutter/infinite_carousel) - A looping, center-focused carousel with fluid depth motion and original illustrated cards.
- [Folder Explosion](flutter/folder_explosion) - A folder icon that bursts into scattered file cards and collapses back on tap.
