# SwiftUI Apple Music Album Expansion

A self-contained Apple Music-style album expansion animation for Motion & Code.

## What It Shows

- A compact album card that expands into a fullscreen music player
- `matchedGeometryEffect` transitions for the card surface, album art, title, and artist
- Album art that scales smoothly between states
- Animated playback progress and equalizer bars
- Auto-playing expand and collapse loop for easy recording

## How To Run

```sh
swift run
```

Or open the folder in Xcode and run the `AppleMusicAlbumExpansion` target.
