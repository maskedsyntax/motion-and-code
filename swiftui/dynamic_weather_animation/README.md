# SwiftUI Dynamic Weather Animation

A self-contained SwiftUI weather animation demo for Motion & Code.

## What It Shows

- A glassy weather card with animated condition and temperature text
- Moving storm clouds with subtle parallax
- Rain particles that run continuously
- Periodic lightning flashes with screen glow
- Auto-playing weather intensity changes for easy recording

## How To Run

```sh
swift run
```

Or open the folder in Xcode and run the `DynamicWeatherAnimation` target.
