# Flutter Dynamic Island Clone

A self-contained Flutter animation demo for Motion & Code.

## What It Shows

- A compact black Dynamic Island-style pill
- Smooth expansion into a live flight tracking notification
- Animated icon glow, progress bar, corner radius, size, and content transition
- Tap-anywhere replay for easy short-form screen recording

## How To Run

```sh
flutter pub get
flutter run
```

For web preview:

```sh
flutter run -d chrome
```
