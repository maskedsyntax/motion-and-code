# Flutter Folder Explosion

A self-contained folder burst animation for Motion & Code.

## What It Shows

- A closed folder shell that opens on tap
- Staggered file cards that scatter outward with depth and rotation
- A warm paper-like backdrop with subtle motion lines and particles
- A second tap collapses everything back into the folder

## How To Run

```sh
flutter pub get
flutter run
```

For web preview:

```sh
flutter run -d chrome
```
