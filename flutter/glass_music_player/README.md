# Flutter Glass Music Player

A self-contained glassmorphism mini player demo for Motion & Code.

## What It Shows

- Frosted translucent player panel with blur and soft border glow
- Custom-painted album art
- Animated play/pause button with pulse feedback
- Progress bar with gradient fill and glowing scrub head
- Slow ambient background gradient for short-form recording
- Auto-play loop plus tap-to-toggle playback

## How To Run

```sh
flutter pub get
flutter run
```

For web preview:

```sh
flutter run -d chrome
```
