# Flutter App Store Download Button

A self-contained Flutter animation demo for Motion & Code.

## What It Shows

- App Store-style `GET` and `OPEN` button states
- Loading spinner, circular progress, and installed checkmark
- Smooth button width changes and content transitions
- Auto-replay plus tap-anywhere replay for short-form recording

## How To Run

```sh
flutter pub get
flutter run
```

For web preview:

```sh
flutter run -d chrome
```
