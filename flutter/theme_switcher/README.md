# Flutter Theme Switcher

A self-contained Flutter animation demo for Motion & Code.

## What It Shows

- A tactile light/dark toggle
- Sun-to-moon transformation
- Animated day/night sky transition
- Stars, clouds, horizon lighting, and reactive UI surface
- Tap or drag interaction with auto-replay for recording

## How To Run

```sh
flutter pub get
flutter run
```

For web preview:

```sh
flutter run -d chrome
```
