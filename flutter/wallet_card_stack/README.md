# Flutter Wallet Card Stack

A self-contained Flutter animation demo for Motion & Code.

## What It Shows

- Apple Wallet-style stacked cards
- Tap to expand or collapse the stack
- Tap a card to open a focused detail view
- Animated depth, shadows, shimmer, and transaction details
- Works well as a vertical short-form recording

## How To Run

```sh
flutter pub get
flutter run
```

For web preview:

```sh
flutter run -d chrome
```
